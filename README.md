# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project focused on analyzing customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI** to uncover meaningful business insights and customer trends.

---

# 📌 Project Overview

This project analyzes customer shopping behavior using transactional data from **3,900 purchases** across various product categories.

The goal of this project is to:
- Analyze customer spending patterns
- Understand customer segments
- Identify top-performing products
- Study discount and subscription behavior
- Build an interactive Power BI dashboard

---

| Feature | Details |

# 🧹 Data Cleaning & Preprocessing

Performed extensive preprocessing using **Python & Pandas**:

✅ Checked missing values  
✅ Imputed missing review ratings using category median  
✅ Converted columns into snake_case  
✅ Created engineered features:
- `age_group`
- `purchase_frequency_days`

✅ Removed redundant columns  
✅ Loaded cleaned dataset into PostgreSQL database  

---

# 🗄️ PostgreSQL Integration

Connected Python with PostgreSQL using SQLAlchemy and loaded the cleaned dataframe into the database for SQL analysis.

## 🔗 Database Connection

```python
from sqlalchemy import create_engine
from urllib.parse import quote_plus

username = "postgres"
password = quote_plus("YOUR_PASSWORD")

engine = create_engine(
    f"postgresql+psycopg2://{username}:{password}@localhost:5432/postgres"
)
```

# 📊 SQL Business Analysis

Performed multiple business-driven SQL analyses:

## 🔹 Revenue by Gender
Compared total revenue generated by male vs female customers.

## 🔹 High-Spending Discount Users
Identified customers who used discounts but still spent above average.

## 🔹 Top Rated Products
Found products with the highest average ratings.

## 🔹 Shipping Type Comparison
Compared average purchase amount between Standard and Express shipping.

## 🔹 Subscribers vs Non-Subscribers
Compared:
- Total revenue
- Average spend
- Customer count

## 🔹 Discount-Dependent Products
Identified products most dependent on discounts.

## 🔹 Customer Segmentation
Segmented customers into:
- New
- Returning
- Loyal

## 🔹 Revenue by Age Group
Analyzed which age groups contributed most revenue.

---

# 📈 Key Insights

✅ Male customers generated significantly higher revenue than female customers.

✅ Express shipping customers had a slightly higher average purchase value.

✅ Products like **Gloves, Sandals, and Boots** received the highest average ratings.

✅ Loyal customers formed the majority customer segment.

✅ Young Adults generated the highest total revenue among all age groups.
